# DGT-studio - Премиальный тюнинг автомобилей

Сайт для премиального центра тюнинга автомобилей, специализирующегося на выхлопных системах, тормозах, кованых дисках и комплексных проектах.

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - Типизация
- **CSS Modules** - Стили
- **Yandex.Metrika** - Аналитика

## Структура проекта

```
dgt-nextjs/
├── app/                      # Страницы приложения (App Router)
│   ├── layout.tsx           # Корневой layout с Header и Footer
│   ├── page.tsx             # Главная страница
│   ├── globals.css          # Глобальные стили
│   ├── exhaust/             # Страница выхлопных систем
│   ├── brakes/              # Страница тормозных систем
│   ├── wheels/              # Страница кованых дисков
│   ├── suspension/          # Страница подвески
│   ├── projects/            # Страница проектов
│   ├── privacy/             # Политика конфиденциальности
│   └── terms/               # Условия использования
├── components/              # Компоненты
│   ├── Header.tsx           # Шапка сайта
│   ├── Footer.tsx           # Подвал сайта
│   ├── HeroSlider.tsx       # Слайдер на главной
│   ├── ServicesSection.tsx  # Секция услуг
│   └── ProjectsSection.tsx  # Секция проектов
├── public/                  # Статические файлы
├── package.json
├── tsconfig.json
├── next.config.js
└── tailwind.config.js
```

## Установка и запуск

### 1. Установка зависимостей

```bash
npm install
```

### 2. Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по адресу [http://localhost:3000](http://localhost:3000)

### 3. Сборка для продакшена

```bash
npm run build
```

### 4. Запуск продакшен версии

```bash
npm start
```

- Автоматическая смена слайдов каждые 6 секунд

## Деплой

### Vercel (рекомендуется)

1. Загрузите проект на GitHub
2. Подключите репозиторий в [Vercel](https://vercel.com)
3. Vercel автоматически определит Next.js и выполнит деплой

### Netlify

```bash
npm run build
```

Загрузите папку `.next` и `public` в Netlify

### Собственный сервер

```bash
npm run build
npm start
```

Запустите на порту 3000 или настройте через переменную окружения `PORT`

## Переменные окружения

Создайте файл `.env.local`:

```env
# Опционально: укажите публичный URL для правильной генерации ссылок
NEXT_PUBLIC_SITE_URL=https://dgt-studio.ru
```
